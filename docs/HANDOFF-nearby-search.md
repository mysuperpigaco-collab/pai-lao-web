# ส่งมอบ: ค้นหา "ใกล้ฉัน" + โหลดการ์ดเพิ่ม (ExplorerSection)

ฟีเจอร์: เพิ่มแท็บ "ใกล้ฉัน" ในกล่อง "เจาะลึกรายพื้นที่" หน้าแรก — ขอ GPS, ปักหมุดลากได้ (จำกัดในไทย),
รัศมี 500ม./1/3/5 กม., หมวดครบ 10, ลิสต์การ์ดใต้แผนที่ + โหลดเพิ่ม (infinite + ปุ่ม) และทำโหลดเพิ่มในแท็บเลือกพื้นที่ด้วย
หยุด auto-load เมื่อหัวข้อ "ไฮไลต์สถานที่" เข้าจอ

## ไฟล์ที่เปลี่ยน

**สร้างใหม่**
- `app/api/places/nearby/route.ts` — GET `?lat=&lng=&radius=&category=` คืนสถานที่ APPROVED ในรัศมี
  (bbox prefilter + haversine จาก `lib/geo.ts`), จำกัดในไทย, รัศมีรับเฉพาะ 500/1000/3000/5000, เรียงตามระยะ, คืนสูงสุด 300
- `components/maps/NearbyMap.tsx` — Leaflet/OSM หมุดลากได้ + วงรัศมี + หมุดผลลัพธ์ (client only, `ssr:false`)

**แก้ไข**
- `components/home/ExplorerSection.tsx` — เพิ่ม two-tab toggle, แท็บ "ใกล้ฉัน", การ์ดผลลัพธ์ + โหลดเพิ่มทั้งสองแท็บ
- `app/page.tsx` — เพิ่ม `id="explore-places-heading"` ที่หัวข้อ "ไฮไลต์สถานที่" (ใช้เป็นจุดหยุด auto-load)

ไม่มีการแก้ schema / ไม่ต้อง `prisma db push` / ไม่เพิ่ม dependency (reuse `useInfiniteScroll`, react-leaflet ที่มีอยู่)

## วิธีรัน / ตรวจ

```bash
npm run dev        # เปิด http://localhost:3000  (geolocation ใช้ได้บน localhost)
npm run build      # ยืนยันว่า build ผ่าน (ผมยังรัน build ฝั่งผมไม่ได้ — รบกวนตรวจตรงนี้)
npx tsc --noEmit   # typecheck เฉพาะ (ถ้าต้องการ)
```

## เช็กลิสต์ทดสอบบนหน้าแรก (เลื่อนลงไปที่ "เจาะลึกรายพื้นที่")

1. สลับแท็บ "เลือกพื้นที่" ↔ "ใกล้ฉัน" ได้
2. แท็บใกล้ฉัน: กด "ค้นหาใกล้ฉัน" → เบราว์เซอร์ขอสิทธิ์ตำแหน่ง → ปักหมุด + วงรัศมี + การ์ดใต้แผนที่
3. ลากหมุด → ผลลัพธ์อัปเดต (หมุดไม่เด้งกล้อง); เปลี่ยนระยะ → แผนที่ซูมตามวงรัศมี
4. เปลี่ยนหมวด → กรองผลถูกต้อง
5. ผลเยอะ: เลื่อนลงโหลดเพิ่มเอง + ปุ่ม "โหลดเพิ่ม" บอกจำนวนที่เหลือ; ครบแล้วขึ้น "แสดงครบ…"
6. **จุดสำคัญ**: เลื่อนลงจน "🗺️ ไฮไลต์สถานที่" โผล่ → auto-load หยุด (ปุ่มยังกดเองได้); เลื่อนกลับขึ้น → auto-load ทำงานต่อ
7. แท็บเลือกพื้นที่: เลือกจังหวัด → โหลดเพิ่มทีละ 12 ทำงานเหมือนหน้า `/place`
8. ตำแหน่งนอกไทย → วางหมุดที่กรุงเทพฯ พร้อมข้อความเตือน ให้ลากเอง

## หมายเหตุ / จุดที่อาจต้องปรับ

- ถ้าตั้ง `NEXT_PUBLIC_ENABLE_MAPS=false` หน้า place จะซ่อนแผนที่ แต่แท็บใกล้ฉันยังโชว์แผนที่อยู่ (ฟีเจอร์นี้พึ่งแผนที่เป็นหลัก) — ถ้าต้องการให้เคารพ flag นี้ บอกได้ ผมเพิ่มเงื่อนไขให้
- รัศมี 5 กม. ในเมืองหนาแน่นอาจคืนหลักร้อยจุด — เบาอยู่ (haversine บนหลักพันแถว <30ms) แต่ถ้าข้อมูลโตมากค่อยพิจารณา index บน `Place.lat,lng`
- ขอบเขตไทยใช้ค่าเดียวกันสองที่: `app/api/places/nearby/route.ts` (`TH_BOUNDS`) และ `components/maps/NearbyMap.tsx` (`TH`) — แก้ต้องแก้ให้ตรงกันทั้งคู่
